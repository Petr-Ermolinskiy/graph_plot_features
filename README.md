# Построение графа корреляционной связи переменных

В данном проекте представлен инструмент для визуализации корреляционных зависимостей между переменными в виде графа.

## Установка UV (менеджер зависимостей)

Установите UV:
```bash
pip install uv
```

Установите все зависимости: 
```bash
uv sync
```
